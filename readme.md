ProstoMesto
=============================
Это учебный проект Mesto,
выполненный в ходе обучения в Яндекс.Практикуме
на факультете фронтенд-разработки

АДРЕС
-----------
- https://prostomesto.tk
- https://bimimot.github.io/ProstoMesto/


ОПИСАНИЕ
-----------
На странице проекта выводятся карточки, сохраненные пользователям:
фото какого-нибудь места или города с подписью и количеством лайков.
Для размещения карточек нужна регистрация.
Зарегистрированные пользователи проекта могут:
- создавать свои карточки,
- удалять своие карточки,
- лайкать любые карточки,
- редактировать данные своего профиля,
- редактировать свою аватарку

РЕШЕННЫЕ ЗАДАЧИ
-----------
- отзывчивая верстка для разных разрешений экрана
- соблюдены принципы ООП - код разбит на отдельные модули (классы)
- сделаны попапы: увеличение фотографий, добавление карточки, авторизация, регистрация, редакироватние профиля и аватара
- попапы закрываются по ESC, крестики и клику мимо попапа
- методология БЭМ для наименования классов CSS 
- кастомная валидация полей в попапах
- реализовано взаимодействие с API
- настроены две сборки: build и dev
- для сборки dev настроено атоматическое обновление hot reload
- для сборки build настроен веб-пак
- вебпак настроен для оптимизации изображения и подгрузки шрифтов
- настроено хэширование CSS и JS файлов 
- настроена транспиляция кода при помощи Babel'я
- проект размещен на том же сервере, что и бэкенд проекта
